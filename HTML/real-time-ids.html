Architecture (Conceptual)

In the full implementation, this dashboard would connect to a backend that performs:

  • Packet capture using libpcap / npcap
  • Flow feature extraction (e.g., CICFlowMeter-style features)
  • Inference using:
    • Supervised classifier (Random Forest) for known attacks
    • Anomaly detector (Autoencoder) for unknown threats
  • Indexing alerts and flow summaries into Elasticsearch
  • Serving this dashboard via REST/JSON APIs

This page focuses on the front-end structure, ready to be wired to real APIs.
